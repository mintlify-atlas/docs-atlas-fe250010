---
title: "Kinoko: MapleStory v95 server emulator"
description: "Kinoko is an open-source MapleStory v95 server emulator in Java. Runs a full Login, Central, and Channel node stack with SQLite or Cassandra."
---

Kinoko is a feature-complete server emulator for MapleStory version 95 (GMS/BMS locale 8). It reproduces the game's multi-node server architecture — Login, Central, and Channel nodes — using Netty for high-performance networking.
