Kinoko is a feature-complete server emulator for MapleStory version 95 (GMS/BMS locale 8). It faithfully reproduces the game's multi-node server architecture — Login, Central, and Channel nodes — using Netty for high-performance networking. Whether you want to run a private server for nostalgia or study classic MapleStory game mechanics, Kinoko gives you a solid, configurable foundation.

## What Kinoko emulates

Kinoko targets **MapleStory v95** and covers the full breadth of that era's content:

- **All job branches** — Explorers, Cygnus Knights, Aran, Evan, and Resistance classes (Battle Mage, Wild Hunter, Mechanic)
- **Complete world systems** — guilds, alliances, parties, friends, messengers, and expeditions
- **Cash Shop** — configurable commodity listings and NX currency
- **NPC scripting** — Java-based annotated script handlers for NPCs, portals, quests, and events
- **Field events** — Continental ship routes, airport flights, elevator events, and subways
- **Persistent storage** — character data, quests, skills, items, and guild records saved to SQLite or Cassandra

## Prerequisites

Before running Kinoko you need:

- **Java 21** (JDK) and **Maven** for building from source, or **Docker** for the containerized path
- **MapleStory v95 WZ files** — the client data files Kinoko reads at startup (see [Quickstart](/quickstart) for the full list)
- Optional: **CassandraDB 5** or **ScyllaDB** if you prefer a distributed database over the default SQLite

<Note>
  Kinoko does not redistribute WZ files. You must provide your own copy of the MapleStory v95 client data.
</Note>
